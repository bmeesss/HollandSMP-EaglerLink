About Us

What is hollandSMP?

hollandSMP is a survival multiplayer server for Minecraft 1.12.2, built on Eaglercraft. That means you don't even need the original Minecraft launcher: the server runs entirely in your browser, on both desktop and mobile.

We deliberately keep the server on the classic 1.12.2 version because it offers the best balance between browser performance and access to the familiar mods, plugins and items many players are nostalgic for.

Survival & Building

Gather resources, build your own base, and work together or go solo on big projects.

Economy & Shop

Earn and spend money through our in-game shop, marketplace, and trading with other players.

Anti-cheat

We run anti-cheat software to keep the server fair for every player.

Active Community

Chat on Discord, report bugs, suggest features you'd like to see, and join in on events.

Server Rules

These rules apply to everyone, in-game and on Discord.

  • 01Respect each other. No bullying, discrimination, or hate speech.
  • 02No griefing or stealing. Stay away from other players' builds and items without permission.
  • 03No cheats or exploits. Using hacks, x-ray, or dupe bugs results in a permanent ban.
  • 04No spam or advertising. No unsolicited ads for other servers.

Questions

Frequently Asked Questions

Do I need to buy Minecraft to play?

No. hollandSMP runs on Eaglercraft, a browser version of Minecraft. You don't need an account or a purchase, just a browser.

Does the server work on mobile?

Yes, use the Mobile Portal above. The server is playable on phone and tablet through the browser.

Is hollandSMP free?

Playing on the server is completely free. Some cosmetic extras are optionally available through the shop.

How do I report a bug or a player?

Report it in our Discord server in the designated channel so staff can review it.
